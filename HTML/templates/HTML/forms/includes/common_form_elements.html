To build an effective form, it is important to use the right input elements, and the are many to choose from in HTML. A full list of them can be found here.

Most of them exist as different values that you can assign to the type attribute of the <input> element. Please note that the input element is self closing. Listed here are a few useful ones with examples:

Text Input

Used for entering short text values, it provides a text input field, accepts letters, numbers, special characters.

{% include 'HTML/forms/includes/code_examples/input_text_input.html' %}

Password Field

Hides characters entered by the user by replacing any entered characters with *.

{% include 'HTML/forms/includes/code_examples/input_password_field.html' %}

Email Input

Automatically validates email format to check that the entered value at least looks like a valid email address.

{% include 'HTML/forms/includes/code_examples/input_email_input.html' %}

Search Input

Similar to text input but has an X that comes up when a query is entered. This X clears the entered data (as does pressing esc).

{% include 'HTML/forms/includes/code_examples/input_search_input.html' %}

Number Input

For number inputs, with an up/down arrow to help users pick a value. Use the min and max attributes to define the minimum and maximum values respectively.

{% include 'HTML/forms/includes/code_examples/input_number_input.html' %}

Range Input

A slider that can be controlled with arrow keys, also uses the min and max attributes.

{% include 'HTML/forms/includes/code_examples/input_range_input.html' %}

Radio Buttons

Allows users to select one option from multiple choices. The radio group is defined by giving each of the buttons the same name attribute.

{% include 'HTML/forms/includes/code_examples/input_radio_buttons.html' %}

Check Boxes

Lets users select one or multiple options. The value attribute dictates what is sent to the server in the value: input pair. To have an option pre-checked, add the checked attribute.

{% include 'HTML/forms/includes/code_examples/input_check_boxes.html' %}

Date Selector

Creates an input field that shows a date-picker when clicked.

{% include 'HTML/forms/includes/code_examples/input_date_selector.html' %}

Select

Select is its own element rather than a value for the type attribute of an input element and has a closing tag associated with it. Within the element, there are option elements for each choice available to the user. Select creates a drop-down menu for the user to pick an option from. Add the selected attribute to have that option selected by default.

{% include 'HTML/forms/includes/code_examples/input_select.html' %}

Textarea

Textarea is also its own element rather than a value for the type attribute of an input element and has a closing tag associated with it. It gives the user an area to enter more text. It can have attributes of rows and cols to format the height and width respectively and you can give the placeholder attribute a value to fill it with placeholder text or prepopulate it with text by typing it between the <textarea></textarea> tags.
