Links can be customised depending on their state. These can be seen when using links on the web. Have you ever clicked a blue link, only to go back to the page and see it is now purple? This is because there is a rule for the :visited link state to highlight to the user that the link has already been clicked. These states are pseudo classes and the selectors for them follow the syntax of:

{% include 'CSS/links_and_buttons/includes/code_snippets/link_state.html' %}
Pseudo-class Description
:link Styles an unvisited link.
:visited Styles a visited link.
:hover Styles the link when the mouse is hovering over it.
:active Styles the link at the moment it is being clicked.
