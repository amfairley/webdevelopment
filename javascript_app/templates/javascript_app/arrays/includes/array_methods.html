JavaScript provides numerous methods to manipulate arrays.

concat(): Merges arrays

{% include 'javascript_app/arrays/includes/code_snippets/methods_concat.html' %}

filter(): Iterates over an array and filters to only a specified set of results. The call back function must return a test that evaluates to be true or false. If the callback function is advanced, define it outside of the function. Filter will return an array. If you are seeking only 1 value, add [0] after filter() to return one object or value instead of an array with 1 item.

{% include 'javascript_app/arrays/includes/code_snippets/methods_filter.html' %}

indexOf(item, fromIndex): Returns the first index of the item.

{% include 'javascript_app/arrays/includes/code_snippets/methods_indexof.html' %}

lastIndexOf(item, fromIndex): Returns the last index of the item.

{% include 'javascript_app/arrays/includes/code_snippets/methods_lastindexof.html' %}

length: Returns the number of elements in the array.

{% include 'javascript_app/arrays/includes/code_snippets/methods_length.html' %}

map(): Takes a callback function as the parameter and map() iterates over an array, applying this function, and returns a new array of results.

{% include 'javascript_app/arrays/includes/code_snippets/methods_map.html' %}

pop(): Removes the last element.

{% include 'javascript_app/arrays/includes/code_snippets/methods_pop.html' %}

push(): Adds one or more elements to the end.

{% include 'javascript_app/arrays/includes/code_snippets/methods_push.html' %}

reduce(): Reduces all elements of an array into a single output value.

{% include 'javascript_app/arrays/includes/code_snippets/methods_reduce.html' %}

reverse(): Reverses the array.

{% include 'javascript_app/arrays/includes/code_snippets/methods_reverse.html' %}

shift(): Removes the first element and shifts all element indexes one to the left.

{% include 'javascript_app/arrays/includes/code_snippets/methods_shift.html' %}

slice(start, end): Returns a shallow copy of a portion of an array.

sort(): Sorts the elements in various ways. By default it sorts alphabetically.

splice(start, deleteCount, item1, item2, ...): Adds/removes elements.
