Syntax Event it Tracks
DOMContentLoaded The DOM content has finished loading but external assets like images, JS files, CSS files etc have not loaded yet.
onload When the page or an element (like an image) has finished loading.
onunload When the user has left the page.
onresize When the window is resized.
onscroll When the user scrolls the page or an element.
onbeforeunload When the user is about to leave a page. This can be used to show a confirmation/warning box.
