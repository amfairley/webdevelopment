Symbol Meaning Example
< Less than. x<10;
<= Less than or equal to. x<=10;
> Greater than. x>5;
>= Greater than or equal to. x>=6.
=== Strict equality. x==="5";
!== Strict inequality. x!=="5";
&& Logical AND. A&&B;
! Logical NOT. !A

Always use === and !== instead of == or != to avoid type coercion bugs.
